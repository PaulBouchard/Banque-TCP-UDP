### Projet Réseaux : Gestion de comptes bancaires 
Date: 22/12/2023

## Description
L'objectif est d'implémenter une application permettant de gérer des comptes bancaires en utilisant une architecture de type client-serveur.

## Compilation
1. Ouvrir un terminal
2. Se rendre dans le répertoire correspondant à l'aide de la commande "cd"
3. Ecrire la commande "make" pour compiler le code

## Execution
1. Ouvrir un second terminal
2. Se rendre aussi dans le répertoire correspondant à l'aide de la commande "cd"
3. Ecrire "./server" dans un terminal, et ENSUITE "./client" dans le second
